# touhidur.bd

Personal site of Md. Touhidur Rahman — Backend & DevOps Engineer.

## Stack

- React 19 + TypeScript 6
- Vite 8
- Bun (package manager)
- Tailwind CSS 4 (via `@tailwindcss/vite`)

## Develop

```bash
bun install
bun run dev      # local dev server
bun run build    # type-check + production build (dist/)
bun run preview  # preview the production build
```

## Edit content

All text, links, roles, projects, and stats live in `src/data.ts` —
update that one file and the whole site follows.
